咖啡历史故事

从教堂后院的一株树，到吧台上的一杯云南豆。

咖啡进入中国的叙事常从云南开始：19 世纪末，传教、商路和边地交流把咖啡带到西南山地；1904 年前后，法国传教士田德能在云南宾川朱苦拉一带种下咖啡苗，村民后来把咖啡采摘、铁锅古法烘焙、石臼捣碎、陶罐煮饮的习惯保留下来。再往后，保山、普洱、德宏、临沧等产区把咖啡从村落记忆推向产业，本土精品豆也开始进入独立咖啡馆和连锁咖啡馆菜单。

城市菜单

本土精品进入吧台。

今天的独立咖啡馆和连锁咖啡馆会把云南豆用于手冲、SOE、拿铁和生椰拿铁，消费者开始把“中国咖啡”当作可被选择的风味。

中国咖啡馆常用豆种。

吧台常见选择通常分两类：稳定出品的意式拼配豆，和用于手冲、特调的单一产区豆。

本土产区

云南卡蒂姆

坚果、可可、低酸和烘烤甜感明显，价格和稳定性适合日常美式、拿铁和冷萃。

常见：美式 / 拿铁 / 生椰拿铁
经典品种

铁皮卡 / 波旁

干净、甜感细、酸质柔和，常被咖啡馆用于展示云南精品豆或单品豆的层次。

常见：手冲 / 冰滴 / 精品美式
拼配基底

巴西

坚果、巧克力和低酸感稳定，是很多中国咖啡馆意式拼配里的厚度来源。

常见：浓缩 / 拿铁 / 摩卡
均衡甜感

哥伦比亚

焦糖、红果和均衡酸甜适合做单品，也常和巴西、云南豆一起进入拼配。

常见：美式 / 澳白 / 手冲
明亮香气

埃塞俄比亚

花香、柑橘、莓果和茶感突出，适合需要“香气记忆点”的手冲和果咖特调。

常见：手冲 / 橙C美式 / 冷萃
厚重低酸

曼特宁

草本、黑巧、香料和厚重口感明显，适合做浓郁黑咖和奶咖基底。

常见：深烘美式 / 拿铁 / Dirty
高酸结构

肯尼亚

黑醋栗、番茄和明亮酸质辨识度高，常用于想突出果酸结构的单品菜单。

常见：手冲 / 冰美式 / 果咖
门店拼配

意式综合豆

通常用巴西、哥伦比亚、云南、印尼或埃塞做比例组合，目标是稳定油脂、甜感和奶咖融合度。

常见：全店基础浓缩

不同咖啡做法，看懂变量和口感。

一杯咖啡不是只有配方名，研磨度、粉水比、萃取率、水温、液重、奶泡、冰量和橙汁比例都会改变酸感、甜感、苦感、厚度与余韵。

清爽果咖

橙C美式

浓缩、冷萃或冰美式做基底，加入橙汁、橙片和冰块。适合埃塞、肯尼亚、浅烘云南豆这种有柑橘酸和花果香的豆子。

结构：咖啡 + 橙汁 / 橙味基底 + 冰水
  • 橙汁比例越高，酸甜和果香越明显，但咖啡余韵会被削弱。
  • 冰量越多，入口更清爽；冰太少会显得甜腻、酸感集中。
  • 萃取率偏低会尖酸，偏高会让橙皮感变苦。
查看详细步骤
  1. 准备杯底：杯中加入冰块至七分满，倒入 80-110g 橙汁或橙味基底。
  2. 处理咖啡：用 18g 粉萃取 32-38g 浓缩，想更清爽可换成 90-120g 冷萃。
  3. 控制分层：沿杯壁或勺背缓慢倒入咖啡，保留上层深色层次。
  4. 校准口感：如果橙汁太甜，增加 20-30g 冰水；如果咖啡发苦，下次把研磨略调粗。
  5. 出杯收尾：用橙片或橙皮点香，喝前轻搅两下，让酸甜和咖啡余韵融合。
黑咖基础

美式

浓缩加水，目标是降低浓度但保留香气。它会放大豆子的干净度，所以稳定拼配或甜感清楚的单品更适合。

结构：浓缩 + 热水 / 冰水
  • 液重越高，浓缩越稀释，苦感降低但厚度变薄。
  • 水温热美式更放大苦甜，冰美式更强调酸感和清爽度。
  • 研磨度太粗会酸薄，太细会苦涩、余韵发干。
查看详细步骤
  1. 选择杯型：热美式先温杯，冰美式加冰至七分满，准备 120-180g 水。
  2. 萃取浓缩：用 18g 粉萃取 30-42g 浓缩，观察流速是否落在 25-32 秒。
  3. 决定顺序：热饮建议浓缩倒入热水，保留香气；冰饮先水和冰，再倒浓缩快速降温。
  4. 判断味道：酸薄通常是萃取不足或水太多，苦涩发干通常是研磨过细或液重过长。
  5. 记录下次：先闻香再入口，记录液重、水量和研磨度，下一杯只改一个变量。
奶咖日常

拿铁

浓缩加入大量牛奶，奶甜会柔化酸感和苦感。巴西、云南卡蒂姆、曼特宁或意式综合豆更容易做出顺滑甜感。

结构：浓缩 + 牛奶 + 轻奶泡
  • 奶量越多，口感越柔，咖啡香气越弱。
  • 奶温过高会带熟奶味，甜感反而变钝。
  • 萃取率偏高会在奶里表现为焦苦，偏低则像酸奶感。
查看详细步骤
  1. 萃取基底：萃取 32-40g 浓缩，优先选择坚果、可可或焦糖调豆子。
  2. 蒸奶打发：倒入冷牛奶至拉花缸 1/3-1/2，蒸汽棒贴近液面打入少量空气。
  3. 细化奶泡：形成漩涡后把大泡打碎，55-62C 停止，牛奶应像湿油漆一样有光泽。
  4. 融合咖啡：先从高处细流注入，让浓缩和牛奶混合，降低表面色差。
  5. 完成拉花：贴近液面推出图案；若图案散开，说明奶泡太薄或注入太快。
强烈分层

Dirty

冰牛奶上直接接热浓缩，入口先热后冷、先浓后甜。它依赖浓缩厚度和温差，适合油脂稳定、可可坚果调的豆子。

结构：冰牛奶 + 热浓缩
  • 液重短一点更厚重，长一点更柔和但分层感下降。
  • 牛奶温度越低，冷热反差越明显。
  • 杯型窄杯更容易保留分层，宽杯更快融合。
查看详细步骤
  1. 提前冷藏：把玻璃杯和牛奶冷藏，杯中倒入 120-150g 冰牛奶。
  2. 短萃浓缩：建议 18g 粉出 28-34g 液体，让浓缩厚度更集中。
  3. 保留温差：浓缩萃好后立刻倒入，不要等待太久，否则 Dirty 的冷热反差会消失。
  4. 做出分层：让浓缩沿杯壁或勺背落到牛奶表面，减少搅动和混合。
  5. 立即饮用：先感受热浓缩的强度，再让冰牛奶的甜感收尾；放久会变成普通冰拿铁。
细腻奶感

澳白

比拿铁更突出咖啡浓度，奶泡更薄，入口更绵密。适合甜感清晰、苦感干净的中深烘拼配。

结构：双份浓缩 + 少量牛奶 + 薄奶泡
  • 咖啡占比越高，苦甜和烘焙香越突出。
  • 奶泡厚度越薄，口感越丝滑；太厚会接近卡布奇诺。
  • 粉量不足会水感明显，粉量过高会压住奶甜。
查看详细步骤
  1. 准备小杯：选择 150-180ml 杯量，让咖啡浓度保留在前段。
  2. 萃取双份：使用双份浓缩，液重控制在 32-38g，避免拉太长变水。
  3. 薄打牛奶：牛奶只打入极少空气，目标是薄而亮的微泡，不做厚泡层。
  4. 低位融合：先低位注入融合咖啡和奶，再用细流收尾保持表面细腻。
  5. 检查口感：入口应紧致、丝滑、咖啡存在感清楚；像拿铁则奶量过多或奶泡太厚。
经典比例

卡布奇诺

咖啡、牛奶、奶泡更接近三段平衡，口感蓬松，香气更浮在上层，适合想保留咖啡存在感的奶咖。

结构：浓缩 + 牛奶 + 厚奶泡
  • 奶泡越厚，触感越轻盈，液体甜感会变弱。
  • 杯量越小，咖啡浓度越明显，苦甜更集中。
  • 水温 / 萃取率偏高会让奶泡下的咖啡显苦。
查看详细步骤
  1. 萃取浓缩：萃取 30-36g 浓缩，中深烘拼配更容易做出稳定苦甜。
  2. 蒸奶进气：蒸牛奶时比拿铁多打入一点空气，让奶泡有支撑但不粗糙。
  3. 建立泡层：注入时先穿透 crema 融合，再提高拉花缸形成厚泡层。
  4. 控制比例：杯量越小咖啡越突出，奶泡越厚触感越轻，但液体甜感会下降。
  5. 完成判断：表面可撒少量可可粉，入口应轻盈、绵密，且咖啡存在感清楚。
巧克力调

摩卡

拿铁里加入巧克力酱或可可基底，适合低酸、坚果、可可调性的咖啡豆。酸质太高会和巧克力冲突。

结构：浓缩 + 牛奶 + 巧克力
  • 巧克力量越高，甜感和厚度增加，咖啡香会后退。
  • 豆子酸感越高，越容易出现果酸和可可分离。
  • 奶量越少，可可苦和咖啡苦越明显。
查看详细步骤
  1. 做巧克力底：杯底加入 15-25g 巧克力酱或可可糖浆，酸感高的豆子要减少用量。
  2. 萃取并搅匀：萃取浓缩后先与巧克力充分搅匀，避免糖浆沉底。
  3. 打奶到拿铁厚度：牛奶温度控制在 55-62C，奶泡薄而顺滑即可。
  4. 组合饮品：倒入牛奶时先融合，再保留一点表面纹理，让可可香和咖啡苦甜连接。
  5. 调整甜度：过甜时下次减少巧克力或提高咖啡占比；苦涩时缩短液重或换低酸豆。
中式热门

生椰拿铁

用椰乳替代部分牛奶，甜感更外放，适合云南、巴西或意式综合豆做稳定基底。椰香强时要避免过酸豆子。

结构：浓缩 + 椰乳 + 冰块
  • 椰乳比例越高，香气更热带，但咖啡厚度会被盖住。
  • 冰量控制稀释速度，冰多更清爽，冰少更甜腻。
  • 液重短萃更适合椰乳，长萃容易显苦和发空。
查看详细步骤
  1. 准备椰乳：杯中加入冰块，倒入 120-160g 冰椰乳，椰乳越多香气越外放。
  2. 短萃浓缩：使用 18g 粉萃取 30-36g 浓缩，保持厚度和苦甜支撑。
  3. 控制倒入：把浓缩缓慢倒在冰块上，形成咖啡与椰乳的层次。
  4. 平衡口感：如果椰香盖过咖啡，下次减少椰乳或把液重缩短；如果发苦，略调粗研磨。
  5. 出杯饮用：喝前轻搅，让椰香、咖啡苦甜和冰块稀释速度稳定融合。
手冲变量

研磨度 + 粉水比决定骨架。

研磨越细、粉水比越高，浓度和厚度上升，但过度会苦涩；研磨越粗、比例越低，口感更清，但容易酸薄、余韵短。

萃取变量

水温 + 时间影响酸甜苦。

水温高、时间长会提高萃取率，甜感之后苦感也会上来；水温低、时间短更明亮，但不足时会出现尖酸和空洞感。

机器变量

粉量 + 液重决定浓缩表达。

同样 18g 粉，液重 30g 更厚更短，36g 更均衡，42g 更清瘦；萃取时间偏短会酸，偏长会苦涩发干。

奶咖变量

奶量、奶泡和冰量决定口感。

奶量增加会降低咖啡存在感，奶泡增加会让口感更轻，冰量增加会让入口更清爽但也更快稀释甜感。

完美奶泡

蒸汽棒先打发空气，再用漩涡细化。

奶泡成功的核心不是“越厚越好”，而是让牛奶有光泽、流动性和细腻微泡。这样拿铁更顺滑，澳白更绵密，拉花也更容易铺开。

1. 纸巾擦净蒸汽棒先排水，再用纸巾擦净喷嘴，避免冷凝水和残奶影响味道。
2. 先打发喷嘴贴近液面听到轻微进气声，1-3 秒即可，声音尖锐说明进气太猛。
3. 做漩涡把喷嘴略微埋入牛奶侧边，让液面旋转，粗泡会被卷进液体里变细。
4. 控温拉花55-62C 停止，轻敲拉花缸并旋转，得到可流动的细腻微泡。
